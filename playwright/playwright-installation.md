This guide shows how to install Playwright and run your first test.
It assumes no prior Playwright setup.

Follow the steps in order.

---

## Step 1: Install Node.js

Playwright requires Node.js.

### Check if Node.js is installed
Run: node --version and npm --version


If not installed:
1. Go to https://nodejs.org
2. Download **LTS version**
3. Install with default options
4. Restart terminal after installation

Verify again using the commands above.

---

## Step 2: Create a New Project Folder

Create a folder: playwright-project  


Open terminal inside this folder.

---

## Step 3: Initialize Playwright Project

Run: npm init playwright@latest
