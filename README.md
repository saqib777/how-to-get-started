## Overview

This repository is a practical, step-by-step guide to help beginners and aspiring testers get started with commonly used development and test automation tools.

The focus is on **clear instructions**, **real commands**, and **common mistakes**, not theory.  
Each guide is written from hands-on experience and is meant to be followed from a clean machine.

---

## Who this repo is for

- Beginners in software testing
- Manual testers moving into automation
- Students learning QA tools
- Anyone stuck during initial setup
- Engineers who want a quick reference

---

## What this repo covers

This repository documents how to set up and start using:

- Git & GitHub
- Selenium (Python & Java)
- Playwright
- Cypress
- Katalon Studio
- VS Code and supporting tools
- Common dependencies like Node.js and browser drivers

Each section includes:
- Prerequisites
- Installation steps
- Verification steps
- Common issues and fixes

---

## Folder Breakdown

### Git & GitHub
Covers repository creation, basic Git commands, and frequently faced errors during commits and pushes.

### Selenium
Step-by-step setup for Selenium with Python and Java, including browser driver configuration and common failures.

### Playwright
Installation, project creation, and writing the first Playwright test.

### Cypress
Installing Cypress, opening the test runner, and running the first test.

### Katalon
Downloading Katalon Studio and creating the first test case.

### Tools
Supporting tools required for automation such as VS Code, Node.js, and browser drivers.

## Why this repo exists

Getting started is often the hardest part.  
Most tutorials skip small but critical steps, leading to frustration.

This repository exists to:
- remove setup confusion
- save time
- provide a single trusted reference
- help others avoid common mistakes

## How to use this repo

1. Pick the tool you want to start with
2. Open the corresponding folder
3. Follow the guide step by step
4. Use the common issues section if you get stuck
