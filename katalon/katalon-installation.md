This guide shows how to download, install, and run your first test in Katalon Studio.
It assumes no prior Katalon setup.

Follow the steps in order.

---

## Step 1: Download Katalon Studio

1. Go to https://katalon.com
2. Click **Download**
3. Choose **Katalon Studio**
4. Select your operating system (Windows / macOS / Linux)
5. Download the installer or ZIP file

You may be asked to sign in or create a free Katalon account.

---

## Step 2: Install Katalon Studio

### On Windows
- If you downloaded an installer, run it and follow the steps
- If you downloaded a ZIP file, extract it to a folder
- Open `katalon.exe`

### On macOS
- Open the `.dmg` file
- Drag Katalon Studio to Applications
- Launch Katalon Studio
